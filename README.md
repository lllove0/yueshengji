# 中英文阅读打卡

一个本地可运行的中英文阅读打卡系统，包含阅读器、后台管理、用户管理、文本上传、PDF 上传占位、内容编辑和分段打卡。

## 功能

1. 用户登录
2. 角色区分：管理员、编辑、普通用户
3. 中英文阅读器
4. 文本内容上传和编辑
5. PDF 文件上传保存
6. 按段落自动生成打卡任务
7. 用户按阅读段落打卡
8. 今日打卡、本文进度、连续天数统计
9. 浏览器语音朗读
10. 后台用户管理

## 本地运行

首次安装依赖：

```bash
npm install
```

开发模式：

```bash
npm run dev
```

默认访问：

```text
http://127.0.0.1:3000/
```

同一 Wi-Fi 下手机访问：

```text
http://你的电脑局域网IP:3000/
```

## 默认账号

管理员：

```text
admin / admin123
```

普通用户：

```text
reader / reader123
```

## 数据位置

运行后会自动生成本地数据和上传目录：

```text
data/app-db.json
uploads/
```

`data/app-db.json` 保存用户、阅读内容、打卡记录。`uploads/` 保存上传文件。

## 构建与生产启动

```bash
npm run build
npm start
```

## PDF 说明

当前版本支持上传 PDF 并在后台生成一条可编辑内容记录。PDF 自动抽取文本需要接入 PDF 解析库，例如后端加入 `pdf-parse` 或类似工具；接入后可以在 `/api/uploads` 中把 PDF buffer 转成文本，再生成打卡段落。
